## Articles & Writing

Blog posts, essays, and deep-dives on agent loop design, harness engineering, and autonomous execution patterns.

[Loop Engineering](https://addyosmani.com/blog/loop-engineering/) - A comprehensive introduction to the emerging discipline of Loop Engineering — designing, building, and operating autonomous AI agent loops. By @addyosmani.  
[WTF Is a Loop? Peter Steinberger vs. Boris Cherny](https://x.com/mvanhorn/status/2063865685558903149) - A conversation between two engineers who've built real-world AI agent loops, covering what a loop actually is and why it matters. By @mvanhorn.  
[WTF Is a Loop? Part 2: The 15 Loops People Are Actually Running](https://x.com/mvanhorn/article/2068426104088748331) - A catalog of 15 real-world agent loops people are running in production, with the commands to steal them. By @mvanhorn.  
[Loop Engineering](https://x.com/addyosmani/status/2064127981161959567) - Addy Osmani's take on Loop Engineering as a new discipline, with practical examples of autonomous agent loops. By @addyosmani.  
[Loops Explained: Claude, GPT, Mira and What Actually Works](https://x.com/AnatoliKopadze/status/2068328135611822149) - A comparative breakdown of how different AI coding tools handle agent loops and which approaches actually deliver. By @AnatoliKopadze.  
[Loop Engineering: The 14-Step Roadmap from Prompter to Loop Designer](https://x.com/0xCodez/status/2064374643729773029) - A structured progression from basic prompting to designing autonomous agent loops. By @0xCodez.

## Open Source Projects

Frameworks, libraries, and tools for building autonomous agent loops.

[loopy](https://github.com/Forward-Future/loopy) - A framework for building and running autonomous AI agent loops. By @Forward-Future.  
[Loops](https://loops.elorm.xyz/) - A collection of closed-loop workflows for coding agents. Each loop includes triggers, feedback gates, and exit conditions so agents self-pace until the job is done.
